Default values

You can now define a single default value by adding curly brackets around a value like this {default value}. Support for multiple ones will be implemented at some point.

Ranges

If you want to define ranges, create a new custom field with the following name e.g. Price range &pound;(after)

  • range - this is really important and must be present for the range to work correctly
  • &pound; - don't use straight symbols (like € or £) but entity ones e.g. &pound; & &euro; $ and &cent are valid symbols.
  • (after) - don't leave any spaces after e.g. $ and (after). This basically says where to add the measuring unit (be it a currency or something like m3 etc.) - (before) is default.

Ranges are defined 10-20, 21-30 etc. (no measuring units - they get pulled from the custom set name).
